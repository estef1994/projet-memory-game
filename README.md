HTML
CSS
JavaScript

Gestion des événements
Logique de jeu

Le principe est simple :

Le joueur retourne deux cartes.
Si les cartes sont identiques, elles restent visibles.
Sinon, elles se retournent automatiquement.
Le but est de retrouver toutes les paires avec le moins de coups possible.

- Installation

1. Cloner le projet
git clone https://github.com/estef1994/projet-memory-game.git
2. Ouvrir le dossier
cd projet-memory-game
3. Lancer le projet

Ouvrir simplement le fichier index.html dans le navigateur.
